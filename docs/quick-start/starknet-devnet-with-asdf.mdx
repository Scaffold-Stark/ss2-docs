# 🚀 StarkNet Devnet - Installation and Setup with asdf

Este documento proporciona una guía para instalar y ejecutar **StarkNet Devnet** utilizando `asdf`, una herramienta basada en Rust para desarrolladores. A continuación, encontrarás instrucciones para la instalación usando `asdf`.

## 📝 Requirements

- **asdf**: Instala `asdf` siguiendo la [documentación oficial de asdf](https://asdf-vm.com/).

---

## ⚙️ Installation Using `asdf`

1. **Añadir el plugin de StarkNet Devnet**:

   ```bash
   asdf plugin add starknet-devnet
   ```

2. **Listar todas las versiones disponibles**:

   ```bash
   asdf list-all starknet-devnet
   ```

3. **Instalar la última versión**:

   ```bash
   asdf install starknet-devnet latest
   ```

4. **Configurar la versión instalada como global o local**:
   ```bash
   asdf global starknet-devnet <VERSIÓN>
   # o
   asdf local starknet-devnet <VERSIÓN>
   ```

---

## 🏃‍♂️ Running StarkNet Devnet

Después de la instalación, puedes ejecutar StarkNet Devnet con el siguiente comando:

```bash
starknet-devnet
```

## 🛠️ Troubleshooting

Si encuentras algún problema durante la instalación o ejecución, considera lo siguiente:

- Asegúrate de tener la última versión de `asdf` instalada.
- Verifica que el plugin de StarkNet Devnet se haya añadido correctamente.
- Consulta los [issues](https://github.com/starknet/starknet-devnet/issues) en el repositorio de GitHub para soluciones conocidas.
